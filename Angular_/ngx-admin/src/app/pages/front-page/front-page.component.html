Viral Fusion Peptides WebServer

Welcome to the Viral Fusion Peptides WebServer, developed by a joint effort of the Centre of Biological Engineering of University of Minho and ITQB NOVA, in a project funded by FCT (Fundação para a Cieência e Tecnologia), which is "is the Portuguese public agency that supports science, technology and innovation, in all scientific domains, under responsibility of the Ministry for Science, Technology and Higher Education", under the "ViralFP" project.

The aim of this webserver is to act as a centralized datasource for all data relevant for the understanding of viral fusion peptides. The gathered data is mostly originally from sources such as UniProt, NCBI Protein and PDB.

This database contains the sequence, structure, key annotations and databases' entries (and references) for each known fusion protein (page Fusion Proteins).

In the page Fusion Peptides, the user can find the protein's fusion peptide (when known) information, referring its position, sequence and structure.

Finally it also contains the taxonomy data of each protein's enveloped virus (a group that contains HIV, Dengue, Influenza and SARS) (on the page Virus' Taxonomy).

Besides those pages (and all those that can be accessed through them to display any related data), this webserver contains links to several Tools: NCBI BLAST, EBI HMMER, Clustal and Weblogo. Finally, this server has peptide prediction tools: epitope prediction (using IEDB) and Fusion Peptide prediction (using machine learning models trained with Propythia).
